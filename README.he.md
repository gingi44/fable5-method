## התקנה — בחרו דרך אחת

### ▸ ל‑Claude Code (בתוך VS Code / Cursor / טרמינל)

פתחו את הטרמינל (ב‑VS Code או Cursor: **View → Terminal**, או **Ctrl + `**),
הדביקו את השורה הבאה ולחצו Enter:

</div>

```sh
npx fable5-method
```

<div dir="rtl" align="right">

הפעילו מחדש את Claude Code — וזהו, הוא נדלק לבד בכל פעם שאתם בונים או מתקנים משהו.
*(אם המחשב לא מזהה את `npx` — התקינו פעם אחת את Node.js מ‑[nodejs.org](https://nodejs.org).)*

### ▸ ל‑Claude באתר או באפליקציה (בלי טרמינל)

**[⬇ להורדת fable5-method.zip](https://github.com/gingi44/fable5-method/releases/latest/download/fable5-method.zip)** —
ואז בתוך Claude: **Settings → Skills → Add skill → Upload**, ובחרו את הקובץ. זהו ✅

> **משתמשים ב‑AI אחר?** (ChatGPT, Cursor, Gemini, Copilot) — אחרי ההתקנה,
> העתיקו את התוכן של `system-prompt.txt` להוראות המותאמות-אישית של הכלי.
